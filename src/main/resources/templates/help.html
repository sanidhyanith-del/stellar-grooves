⚡ Getting Started

First-Time Setup

1
Create an account at the Sign Up page, then log in.
2
In the Scan Directory panel on the left sidebar, enter the absolute path to your music folder (e.g. /home/user/Music or C:\Users\You\Music).
3
Click Start Scan. A progress bar shows files being imported in real time. The scanner finds .mp3, .flac, and .m4a files and extracts metadata (title, artist, album, year, genre, cover art).
4
Once the scan completes, your tracks appear in the library. Click the play button on any row to start streaming.

Navigating the Dashboard

The main dashboard has two panels. The left sidebar contains your playback queue, playlists, and the scan directory form. The right panel is your music library with browsing views for artists, albums, and tracks. Use the stat cards at the top to quickly jump to filtered views (e.g. click "Duplicated Songs" to review duplicates).

▶ Playback

Player Controls

The persistent player bar at the bottom of the screen shows the current track with album art, a seek bar, and volume control. Controls include:

  • Play / Pause — click the play button or press Space
  • Seek — drag the progress bar, or use ← / → arrow keys (5-second jumps)
  • Volume — drag the volume slider, or use ↑ / ↓ arrow keys
  • Shuffle — click the shuffle button to randomize track order
  • Next Track — playback auto-advances when a track ends

Crossfade

Click the sparkle button in the player bar to enable crossfade. When active, the last 3 seconds of each track fade smoothly into the next one using dual audio elements — no gap, no click.

Queue

Add tracks to the "Up Next" queue using the + button on each row. Queued tracks play before the regular library order resumes. The queue syncs to the server so it persists across sessions and devices. You can clear the entire queue with one click.

Lock Screen & Notifications

On supported devices, track metadata and album art appear on your lock screen and in system media notifications. Use the lock screen controls to play, pause, skip, and seek without opening the app.

📚 Library Management

Search & Filter

Use the search box for full-text search across title, artist, and album. Combine it with the artist, album, and genre dropdown filters — all filters work together with AND logic.

Browse Views

Switch between three views using the breadcrumb navigation:

  • Artists — lists all artists with album and track counts. Click an artist to drill down.
  • Albums — grid view with cover art thumbnails (toggle between grid and list). Click an album to see its tracks.
  • Tracks — sortable table with columns for title, artist, album, genre, rating, and decade.

Ratings

Rate any track from 1 to 5 stars using the star widget on each row. Click the same star again to clear the rating. You can sort the track list by rating to surface your favorites.

Genre Editing

Tracks are auto-classified by genre during scanning. If a track shows as "Other", click the genre label to reclassify it. Your correction is remembered — future scans for that artist will use your chosen genre.

Bulk Operations

Use the checkboxes on each row (or the "select all" toggle) to select multiple tracks. A bulk action bar appears with options to delete selected tracks or add them to a playlist.

Duplicates

Click the "Duplicated Songs" stat card to review duplicate tracks (matched by title + artist). You can also detect exact file copies via hash-based duplicate detection. Compare duplicates side by side and delete the ones you don't want.

Trash

Deleted tracks go to a 30-day trash instead of being permanently removed. Restore tracks from trash if you change your mind, or empty the trash to free up space.

🎵 Playlists

Creating & Managing

Create a new playlist from the left sidebar by typing a name and clicking the + button. Add tracks using the + button on each track row, or select multiple tracks and use the bulk "Add to Playlist" action.

Reordering

Open a playlist and drag and drop tracks to reorder them. You can also sort by title, artist, album, genre, rating, or year using the column headers.

Sharing

Click the share button on any playlist to generate a read-only public link. Anyone with the link can view (but not modify) the playlist. Revoke the link at any time to make the playlist private again.

Exporting

Export playlists as M3U (for use in other music players) or JSON (for backup). Use the export button in the playlist view.

🔍 Scanning & Import

How Scanning Works

The scanner recursively walks your music directory looking for audio files (.mp3, .flac, .m4a by default). For each file it extracts metadata using the audio file's embedded tags and computes a SHA-256 file hash for duplicate detection.

Files that have already been imported (by file path or by title + artist match) are automatically skipped. Cover art is extracted and stored separately, subject to a per-user storage quota.

Scan Progress

During a scan, a real-time progress bar shows how many files have been imported, skipped, or encountered errors. You can continue using the app while a scan runs in the background.

Scheduled Scans

Set up automatic scanning on a schedule. In the Scan Directory panel, configure a cron expression and directory path. The system checks every 60 seconds and triggers scans when due — great for keeping your library in sync with a folder that gets new music regularly.

Rate Limiting

To prevent overloading the server, there is a 60-second cooldown between scans. The cooldown timer is shown in the scan panel.

💾 Backup & Export

Library Export

Export your full library metadata as JSON or CSV from the library view. This includes all track metadata (title, artist, album, genre, rating, year) but not the audio files themselves.

Full Backup

Create a complete backup of your library as a single JSON file. This includes tracks, playlists, ratings, genre corrections, and file hashes. Restore the backup on any Stellar Grooves instance to recreate your entire library metadata and playlists.

Playlist Export

Individual playlists can be exported as M3U (compatible with most music players) or JSON.

📥 Install the App

Desktop (Chrome / Edge)

Look for the install icon in the browser address bar (a monitor with a down arrow), or open the browser menu and select "Install Stellar Grooves". The app opens in its own window with no browser chrome and gets its own taskbar/dock icon.

Android

In Chrome, tap the three-dot menu and select "Add to Home screen" or "Install app". Stellar Grooves appears as a standalone app on your home screen and in the app drawer.

iPhone / iPad

In Safari, tap the Share button (square with an arrow), then tap "Add to Home Screen". The app launches in full-screen mode from your home screen.

What You Get

  • Standalone window — no address bar or browser tabs
  • Faster loading — static assets are cached by the service worker
  • Lock screen controls — play, pause, skip, and seek from your lock screen or notification shade
  • Offline page — a friendly message if the server is unreachable, instead of a browser error

⚙ Administration

Admin Dashboard

Users with the Admin role can access the admin dashboard from the navbar. The dashboard shows system-wide statistics (total users, files, playlists) and a paginated user management table.

User Management

Admins can view all registered users with their file counts. Deleting a user permanently removes their account and all associated data (music files, playlists, cover art, queue, and genre corrections).

First Admin Setup

An initial admin user can be created on first startup by setting the ADMIN_PASSWORD environment variable. See the project documentation for details.
